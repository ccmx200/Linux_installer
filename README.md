# Linux Installer

一个现代化的 Linux 系统安装器应用，基于 Electron、Vue 3 和 TypeScript 开发，提供直观、美观的用户界面和流畅的安装体验。

## 项目特性

### 🎨 现代化界面

- 基于 Vue 3 + TypeScript 的响应式设计
- 优雅的动画效果和过渡
- 直观的用户交互体验
- 响应式布局适配不同屏幕尺寸

### 🚀 核心功能

- 自动/手动安装模式切换
- 多镜像源选择与自定义镜像支持
- 详细的系统信息展示
- 实时安装进度跟踪
- 安装过程日志查看
- 安全警告与确认机制

### 🔧 技术栈

- **前端框架**: Vue 3 + Composition API
- **类型系统**: TypeScript
- **构建工具**: Vite
- **桌面框架**: Electron
- **状态管理**: Pinia
- **样式处理**: CSS3 + CSS 变量

## 系统要求

### 运行环境

- Windows 10/11 (64位)

### 硬件要求

- 至少 4GB RAM
- 至少 10GB 可用磁盘空间
- 支持 1024x768 或更高分辨率的显示器

## 快速开始

### 安装依赖

```bash
$ pnpm install
```

## 使用指南

### 1. 启动应用

运行构建后的应用程序或使用开发模式启动。

### 2. 选择安装模式

- **自动安装**: 系统会自动下载并安装 Linux 发行版
- **手动安装**: 您可以手动选择和配置安装文件

### 3. 选择镜像源

- 从预设镜像列表中选择一个合适的镜像源
- 或添加自定义镜像源以加速下载

### 4. 确认安装设置

- 查看系统信息和安装详情
- 确认风险警告并开始安装

### 5. 监控安装过程

- 查看实时安装进度
- 查看详细的安装日志
- 等待安装完成
